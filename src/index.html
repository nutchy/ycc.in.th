YCC คืออะไร?

ค่ายที่จะพาน้องๆ มาช่วยกันระดมความคิดสร้างสรรค์ ด้วยการเขียนโค้ดหรือการออกแบบ เพื่อนำมาแก้ปัญหาที่พบเจอในสังคม และได้ร่วมทีมกับเพื่อนๆ ที่สนใจในด้านเดียวกันอีกด้วย

มาร่วมกันสร้างเว็บไซต์ หรือแอปพลิเคชั่นที่จะช่วยแก้ไขปัญหาที่สนใจ ภายในเวลา 3 วัน พร้อมกับพัฒนาฝีมือไปอีกขั้น ด้วยกิจกรรมเวริคช็อปสนุกๆ แบบจัดเต็ม และรับคำแนะนำจากพี่ๆ ผู้เชี่ยวชาญในวงการเทคโนโลยีและการทำโปรดักต์

ค่ายเปลี่ยนวัยมันส์เป็น “นักสร้างสรรค์” ใน 3 วัน

สาขาที่เปิดรับ

Programming

นักเวทมนตร์ผู้ร่ายคาถาและเสกสรรสิ่งใหม่ๆ ในโลกความเป็นจริงด้วยปลายนิ้วสัมผัส สนใจในการแก้ไขปัญหาของสังคมผ่านทาง การเขียนโปรแกรม เพื่อสร้างสิ่งมหัศจรรย์และประโยชน์ให้แก่โลกใบนี้

Design

นักเวทมนตร์ผู้เปี่ยมล้นไปด้วยจินตนาการ สร้างสรรค์โลกใบใหม่ขึ้นมาผ่านปลายปากกา สนใจในการแก้ไขปัญหาของสังคมผ่านทาง การออกแบบกราฟฟิค เพื่อช่วยเหลือเพื่อนมนุษย์ และสร้างโลกที่สวยงาม
